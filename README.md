# 여권 등록 플랫폼

###### 2024.7 (1人 개인 프로젝트)

<br />

## 📌 Summary

**여권 유무를 정부24 스크래핑을 통해 가져오는 시스템**

- 여권정보를 입력하여 이 여권이 유효한 여권인지 알 수 있음.

<br />

## 🤔 Background

정부24 스크래핑을 통해 여권정보를 입력하고 반환하여 실제 존재하는 여권이면 DB에 저장하는 방식

<br />


## 🔨 Technology Stack(s)

Frontend : node.js(React)
<br />
Backend : express.js
